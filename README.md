Prerequisites:

For telescope:

```bash
brew install ripgrep
brew install fd
```

For Go debuging

```bash
brew install delve
```

Install via:

```bash
rm -rf ~/.config/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.local/share/nvim
rm -rf ~/.cache/nvim
git clone git@github.com:pablobastidasv/neovim.git ~/.config/nvim
```
